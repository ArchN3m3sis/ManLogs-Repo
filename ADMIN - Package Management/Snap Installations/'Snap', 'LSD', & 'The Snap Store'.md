
# The Snap Installation Command Overview

- [x]  Install snap to local Zephyrus Desktop

```bash
sudo apt update 
sudo apt install snapd
```

# Installing LSD

- [x]  Install lsd to local Zephyrus Desktop

```bash
sudo snap install lsd
```
